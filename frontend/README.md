
## Instalación y arranque

```bash
# 1. Instalar dependencias
npm install

# 2. Servidor de desarrollo
npm run dev

# 3. Build de producción
npm run build
```




Coloca las imágenes en `public/img/`:

```
public/img/
├── logo-color.jpeg
├── avatar.jpeg
├── cancha-principal.jpeg
├── cancha-1.jpeg
├── cancha-2.jpeg
└── cancha-3.jpeg
```
